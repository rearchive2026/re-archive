# Re-archive (Public)

이 레포지토리는 재건축 데이터의 **정제된 결과물**을 담고 있습니다.

## 🚀 개요
- **목적:** 가공된 재건축 데이터 제공 및 GitHub Pages 호스팅
- **데이터 출처:** [re-archive-data](https://github.com/rearchive2026/re-archive-data) (Private)에서 자동 가공 및 배포됨

## 🛠 구성
- `index.html`: 데이터 요약 페이지
- `*.json`: 정제된 데이터 파일 (민감 정보 제외)

---
*본 레포지토리는 GitHub Actions에 의해 자동으로 업데이트됩니다.*
